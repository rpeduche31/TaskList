This is a [React.js](https://reactjs.org/) project bootstrapped with [`create-react-app`](https://create-react-app.dev/) and [`material-ui`](https://mui.com/).

## Getting Started

First, install dependencies:

```bash
npm install
# or
yarn install
```

Second, run the development server:

```bash
npm run start
# or
yarn start
```
